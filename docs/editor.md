# 编辑器与实时预览

InkStream 用同一个编辑器内核同时承载“写源码”和“看排版”，你的文档始终是纯文本 Markdown，所见即是真实内容，不会被锁进任何私有格式。

## 单内核：纯文本就是真相源

InkStream 没有“富文本编辑区”和“源码编辑区”两套东西，整个编辑器只有一个内核。无论你看到的是带样式的排版还是裸 Markdown 标记，底层保存的永远是同一份纯文本：标题就是 `#`、加粗就是 `**...**`、链接就是 `[文字](地址)`。

这意味着：

- 你随时可以切到源码看到完整原文，不存在“隐藏的格式数据”。
- 文件用任何文本编辑器打开都是一样的内容，不依赖 InkStream 才能读。
- 切换显示方式只是换一种“看”的方式，不会改动文档本身，也不会丢数据。

## 两种显示模式：Source 与 Live Preview

编辑器有两种显示模式，可随时切换：

- **Live Preview（实时预览）**：边写边渲染，标题显示为大字号、加粗显示为粗体、链接显示为可点文字，`#`、`**` 等标记字符隐藏起来。这是 Markdown / richtext 文档打开时的默认模式。
- **Source（源码）**：显示完整原文，所有 Markdown 标记字符都可见，适合精确编辑标记或排查格式。

切换模式不会重建文档，光标位置、选区、撤销历史和滚动位置都会原样保留。

### 三种切换入口

- **快捷键**：`Ctrl+E`。
- **状态栏指示器**：编辑器底部状态栏有一个模式标签（显示 `Live Preview` 或 `Source`），直接点击即可在两种模式间切换。处于 Live Preview 时标签左侧有一个圆点，Source 时无圆点。
- **命令面板**：按 `Ctrl+Shift+P` 打开命令面板，执行“视图：切换渲染模式”。

每个文件单独记住自己的模式选择：你把 A 文件切到 Source、再切到 B 文件，B 不受影响；切回 A 仍是 Source。该记忆在当前会话内有效，关闭标签页即清除。

### 仅对 Markdown 文档生效

显示模式只对 Markdown 和 richtext 文档有意义。对 LaTeX、Typst、Python 等其他语言的文档，状态栏的模式指示器会隐藏，`Ctrl+E` 不产生任何效果。

## 光标所在行自动展开源码

在 Live Preview 模式下，**光标所在的那一行（或选区覆盖的行）会自动显示为源码**，标记字符重新出现；离开该行后又恢复成排版效果。

这让你既能享受排版预览，又能随时精确编辑标记本身。例如：

- 把光标移到某个标题行，会看到该行重新露出前缀 `#`，但字号仍保持标题大小，行高不会跳动。
- 光标进入一段加粗文字，`**` 标记会显现，方便你修改。
- 离开该行，渲染效果立即回归。

## Live Preview 会渲染哪些元素

进入 Live Preview 后，以下 Markdown 元素会就地渲染（光标所在行除外，按上一节规则展开为源码）：

- **标题** H1–H6：按级别显示为递减的大字号。
- **强调**：加粗、斜体；删除线显示为带横线文字。
- **行内代码**：等宽字体 + 浅色底纹。
- **下划线**：`<u>...</u>` 显示为下划线文字。
- **链接**：仅显示链接文字（地址隐藏），带链接色和下划线。在链接上按住 `Ctrl` 点击可用系统默认方式打开外部链接。
- **Wiki 双向链接** `[[...]]`：显示为链接样式（有别名时显示别名）。
- **列表**：无序列表的 `-`/`*`/`+` 显示为圆点项目符号，有序列表保留编号，并带悬挂缩进。
- **引用块**：左侧显示竖条 + 缩进，文字弱化。
- **任务列表** `[ ]` / `[x]`：显示为可点击的复选框，点击即勾选/取消。
- **水平线** `---`：显示为一条横线。
- **图片** `![](...)`：直接显示图片（支持工作区内本地图片与 `https:` 远程图片）。
- **行内公式** `$...$`：显示为渲染后的数学公式。
- **代码块**：带底纹和右上角语言角标。
- **数学 / Typst / LaTeX 公式块**、**表格**：渲染为对应的排版视图（表格为可就地编辑的表格）。

## richtext 工具栏

当文档的语言被设为 **richtext** 时，编辑区顶部会出现一条常驻的格式工具栏，方便不熟悉 Markdown 标记的用户用按钮排版。工具栏只在 richtext 文档显示，其他文档不出现。

工具栏有四个按钮：

- **加粗**（`Ctrl+B`）：把选中文字包成 `**...**`。
- **斜体**（`Ctrl+I`）：把选中文字包成 `*...*`。
- **下划线**（`Ctrl+U`）：把选中文字包成 `<u>...</u>`。
- **链接**（`Ctrl+K`）：把选中文字变成 `[文字](url)`，并选中 `url` 占位符待你输入地址；无选区时插入空模板把光标放在文字位置。

无论用按钮还是快捷键，产出的都是标准 Markdown（下划线为合法 HTML），保存的物理内容仍是纯文本，不含任何私有格式。

注意事项：

- 这些按钮是“插入/包裹”，不解析选区当前是否已加粗，因此重复点击会叠加标记。
- 加粗（`Ctrl+B`）、斜体（`Ctrl+I`）、链接（`Ctrl+K`）这三个快捷键在普通 Markdown 文档中同样可用（编辑器需处于聚焦状态）；下划线 `Ctrl+U` 为 richtext 文档专属。
- 智能粘贴：在 richtext 文档中选中一段文字后，若粘贴的内容是 `http(s)` 网址，会自动把选中文字变成指向该网址的链接。

## 中文输入

编辑器对中文输入法（IME）做了专门处理：在 Live Preview 模式下连续输入中文、长句合成、切换模式或自动保存时，都不会出现吞字、重复字等问题。
