# 学术模式与 Zotero

Academic（学术）模式把 Zotero 文献库、引用插入、参考文献编译和未解析检查放在同一个界面里，写论文时不用频繁切换工具。

## 进入学术模式

InkStream 有三种写作模式（Standard / Academic / Creative），切换不会改变文件格式，只是调整界面布局与功能集。

- 按 `Ctrl+Shift+P` 打开命令面板，输入「学术」或「Academic」，选择「模式：切换到 Academic（学术）」。
- 切换后，三处界面会随之变化：左侧边栏顶部出现「Zotero 文献库」，编辑器上方出现学术工具栏，右侧面板自动切到「引用」标签页。

## 学术工具栏

进入学术模式后，编辑器正上方常驻一条工具栏，按钮从左到右为：

- **引用**：等同于 `Ctrl+Shift+Z`，触发 Zotero 选择器插入引用。
- **脚注**：在光标处插入脚注标记 `[^N]`，并在文末追加对应的脚注定义行，光标自动移到定义处等你填写（编号自动取文档内未用过的最小数字）。
- **参考文献**：下拉菜单，含「插入 / 刷新（默认 GB/T 7714）」以及 GB/T 7714、APA、Vancouver 三种样式（详见下文）。
- **公式**：插入数学公式块。

工具栏上的按钮与命令面板、快捷键走同一套命令，行为一致。

## 用 Zotero 插入引用

前提：本机已运行 Zotero，并安装了 Better BibTeX（BBT）插件。

1. 把光标放到要插入引用的位置。
2. 按 `Ctrl+Shift+Z`（或点工具栏「引用」按钮）。
3. 弹出 Zotero 的引用选择器，挑选一条或多条文献后确认。
4. InkStream 会把选中文献以 `[@citekey]` 的形式插入光标处（多选为 `[@a; @b]`）。

注意事项：

- 在选择器里直接关闭或取消（不选任何文献），编辑器不会有任何改动。
- 一次只能进行一个引用选择。若上一次选择还没完成又再次触发，会提示你先在 Zotero 选择器里完成或按 Esc 取消。
- 若 Zotero 未运行、未安装 Better BibTeX 或操作超时，会弹出明确的错误提示。
- 引用格式会随当前文档语言自动适配：Typst 文档插入 `#cite(<key>)`，LaTeX 文档插入 `\cite{key}`，其余（如 Markdown）保持 `[@citekey]`。

## 从侧边栏文献库插入

学术模式下左侧边栏顶部的「Zotero 文献库」直接列出 Zotero 库中的条目：

- 顶部输入框可按标题、作者或 citekey 过滤。
- 点击任意一条，即在编辑器光标处插入对应的 `[@citekey]`（同样会按文档语言自动适配格式），无需再弹选择器。
- 右上角的刷新按钮可重新拉取文献库。
- 当 Zotero 未运行时，若已配置过 Web API 同步，面板会自动改读本地离线缓存，并在标题旁显示一个离线小图标。

## 引用面板与未解析标红

右侧面板的「引用」标签页汇总当前文档里的所有引用：

- 逐条列出 `[@citekey]`，重复引用合并计数，右侧用小圆角数字标出次数。
- 顶部显示引用总数；存在未解析项时额外显示「N 未解析」。
- **未解析标红**：某个 citekey 在 Zotero 库中找不到时，该条会以红色显示并带警告图标，鼠标悬停提示「未在 Zotero 库中找到此 citekey」。
- 右上角刷新按钮可重新从 Zotero 解析全部 citekey。
- 状态栏右侧也会显示当前文档的引用数量与未解析数量（无引用时不显示）。

提示：在引用尚未完成解析之前，面板不会把任何条目判为未解析，避免误标红。

## 编译参考文献（GB/T 7714 / APA / Vancouver）

参考文献分「先占位、再展开」两步。样式信息以注释标记写进文档本身，可以反复重新展开刷新。

1. 点工具栏「参考文献 ▸ 插入 / 刷新（默认 GB/T 7714）」。文档若还没有参考文献区，会在文末插入「## 参考文献」标题和一个占位标记。
2. 再次点同一项，InkStream 会扫描全文的 `[@citekey]`，从 Zotero 取出文献数据，按样式渲染成参考文献列表，替换占位区域。
3. 若想换样式，直接在下拉里选「GB/T 7714」「APA」或「Vancouver」，即可按该样式就地展开或重排（即使还没有占位也会直接生成）。

注意事项：

- 重新展开是幂等的：多次点击只会刷新同一块参考文献区域，不会重复堆叠。
- 若某些引用在 Zotero 中找不到，展开后会提示哪些 citekey 未找到，其余可解析的文献仍会正常列出。

## Zotero Web API 同步与离线缓存

默认情况下文献库直接连本机 Zotero（BBT）实时读取。配置 Web API 后，可把整个库同步到本地缓存，这样即便 Zotero 没开，文献库和参考文献展开仍可离线使用。

配置位置：打开设置（`Ctrl+,`）▸「Zotero」分区。

1. 在 zotero.org ▸ Settings ▸ Feeds/API 创建一个含「Allow library access」的只读 API Key，userID 显示在同一页面。
2. 在设置里填入 userID 和 API Key，点「保存」。
3. 保存后点「立即同步」做一次增量同步，结果会落到本地缓存，并显示更新/删除条数与库版本。
4. 不再需要时可「清除凭据」。

注意事项：

- API Key 仅保存在本机操作系统的凭据库，不会上传，也不会回显到界面。
- 同步是增量的：每次只拉取上次同步以来的改动。
- 离线时为「在线优先、离线回退」：先尝试连 Zotero，连不上再读缓存；缓存也为空时才报错。
