# 简易模式

如果你只想把 InkStream 当成一个干净的 Markdown 编辑器——写点东西、存成文件，不需要双向链接、知识图谱、Git、Zotero 这些进阶能力，**简易模式**会把界面精简到只剩基础写作，并且**不在你的工作区里创建任何索引文件夹**。

## 它解决什么

InkStream 默认会在打开的工作区里建一个隐藏的 `.inkstream/` 文件夹，用来存全文搜索与双向链接的索引。对重度用户这是必需的；但对只想写点纯文本的轻度用户，它是多余的——简易模式让你彻底关掉它。

## 怎么开

设置 → **通用** → **简易模式** 开关。打开即生效，随时可关。

## 开启后会怎样

**关闭全部高级功能**，界面只留基础写作：

- 侧边栏：隐藏全文搜索、Git 面板、Zotero 文献库、章节-场景树，只保留文件树。
- 右侧面板：只保留大纲，反链与局部图谱隐藏。
- 状态栏：只留文件路径、光标位置、渲染模式，隐藏 Git 分支、字数目标、引用、模式切换。
- 命令面板 / 菜单 / 快捷键：知识图谱（Ctrl+G）、Git Graph、模式切换、插入引用、切换文档语言等高级命令一并隐藏，即使按对应快捷键也不会触发。

**不创建 `.inkstream/` 索引文件夹**：简易模式下打开 / 切换工作区都不会建索引库，反链、未链接提及、图谱查询一律为空。

**保留的基础能力**：纯文本编辑、Live Preview 与 Source 切换、文件树增删改、自动保存、文件导出、阅读模式照常可用——这些都不依赖索引。

## 随时可逆

关闭简易模式后，全部高级功能立即恢复；重新打开工作区时会自动重建索引库。简易模式是一个纯粹的显示 / 行为开关，不会改动你的任何文档。

> 注意：如果某个工作区在关闭简易模式时已经建过 `.inkstream/`，开启简易模式并不会去删除它（避免误删你的数据）。简易模式只保证「此后不再创建、不再写入」。需要的话你可以手动删除该文件夹。
