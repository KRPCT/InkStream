# 阅读模式

InkStream 除了写作，也能当阅读器用。打开 txt、Word（docx）、EPUB 电子书或 PDF，它会切到一个全屏阅读界面：盖住编辑器和两侧面板，隐藏与阅读无关的功能，只留下正文和一条简单的工具栏。

## 支持的格式

- **txt**：纯文本
- **docx**：Word 文档（转换为排版后的正文）
- **epub**：电子书（按章节顺序连续呈现）
- **pdf**：逐页渲染、滚动阅读

## 怎么进入

- **docx / epub / pdf**：直接打开即自动进入阅读模式（它们不可编辑）。打开方式包括：文件菜单 → 打开文件、把文件拖进窗口、或在系统里用 InkStream「打开方式」。
- **txt**：默认仍在编辑器里打开（txt 可编辑）；想换成阅读视图时，用命令面板（`Ctrl+Shift+P`）→「视图：阅读模式」把当前 txt 切过去。

## 工具栏

阅读界面顶部的工具栏只有几个按钮：

- **文体切换**（小说 / 文献）：见下节。
- **字号** −/＋：调节正文大小。
- **配色**：在 亮色 / 护眼（米黄）/ 夜间 三种配色间循环切换，长时间阅读时换护眼或夜间会舒服一些。
- **关闭**：回到编辑器。

## 自动识别小说 vs 文献

打开文档时，InkStream 会根据正文特征判断它更像**小说**还是**文献 / 论文**，并据此切换排版：

- **小说**：衬线字体、较宽的行距、段首缩进、两端对齐，接近纸质书的观感。
- **文献**：无缩进、紧凑行距、较窄的版心，接近论文 / 报告的观感。

判断依据是章回标记（如「第一章」「Chapter 1」）、对白密度、摘要 / 参考文献等线索。判断不准时，点工具栏的文体按钮手动在「小说 / 文献」间切换即可。

## 沉浸与退出

进入阅读模式后，编辑器、侧边栏、右侧面板都被盖住，相关的编辑功能也就用不到了，状态栏折叠成「阅读模式 · 文件名」。底层编辑器并没有卸载，点关闭（或退出阅读）就回到原来的编辑状态，光标和内容都还在。

> 提示：阅读模式会把文件一次性读进内存，超大文档（约 100MB 以上）会被拒绝，以免影响性能。
