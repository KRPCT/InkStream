# 自动更新

InkStream 内置应用内自动更新：启动时静默检查新版本，有更新时弹出非侵入提示，一键下载并重启即完成升级——无需手动去下载页。

## 怎么用

- **自动检查**：每次启动后台静默检查最新版本。有更新时弹出对话框「InkStream X 现已可用」，可选「现在更新」或「稍后」；无更新或离线时不打扰。
- **手动检查**：标题栏 **帮助 → 检查更新**，或命令面板（`Ctrl+Shift+P`）执行「帮助：检查更新」，或 **帮助 → 关于 InkStream → 检查更新**。无更新会提示「已是最新版本」，检查失败（如断网）会明确提示而非误报最新。
- **下载与安装**：点「现在更新」后显示下载进度条；完成后点「重启并安装」即应用新版本（也可稍后手动重启生效）。下载中对话框不可关闭，避免中断。

## 安全

- 每个更新包都经过开发者私钥签名，InkStream 用内置公钥验签后才安装——下载途中被篡改的包会被拒绝。私钥只存在于发布流水线，绝不随应用分发。
- 更新清单（latest.json）与安装包都从项目 [GitHub Releases](https://github.com/KRPCT/InkStream/releases) 获取，全程 HTTPS。
- 检查与下载由应用原生层（Rust）完成，不经网页层，不受网页安全策略影响。

## 注意

- 自动更新仅在正式安装版中生效，开发版不检查（不会报错）。
- 从 **v1.1.2 起**的安装版才内置更新器；更早版本需手动下载一次新版，此后即可自动更新。
- macOS / Windows 安装包未做系统代码签名，首次安装可能有「未知开发者」提示，属正常。
