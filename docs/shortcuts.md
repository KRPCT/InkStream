# 快捷键与命令面板

通过命令面板搜索并执行任意功能，或用键盘快捷键直接操作——无需记住菜单位置即可驱动 InkStream 的全部命令。

## 命令面板

命令面板是 InkStream 的统一入口：所有功能都能在这里搜索到，并显示对应的快捷键。

打开方式：

- 按 `Ctrl+Shift+P` 打开（再按一次或按 `Esc` 关闭）。
- 打开后输入框已预填 `>`，即进入命令搜索模式，直接接着输入关键字即可。

使用方法：

- 直接输入命令名称的关键字进行筛选，支持简体中文模糊搜索（例如输入「图谱」可命中「视图：知识图谱」，输入「保存」可命中「文件：保存」）。
- 最近用过的命令会自动排在列表最前面（最近最常用置顶，最多记忆 10 条，并跨会话保留）。
- 每条命令右侧会显示它的快捷键，可把命令面板当作快捷键速查表使用。

列表内导航：

- `↑` / `↓`：上下移动选中项。
- `Enter`：执行当前选中的命令。
- `Esc`：关闭面板。

## 快速打开文件

- 按 `Ctrl+P` 打开「快速打开」。
- 当你已经打开了一个工作区（文件夹）时，直接输入文件名关键字即可在工作区内模糊查找并打开文件。
- 它与命令面板共用同一个弹层：在这里输入 `>` 同样会切回命令搜索。

## 全局快捷键

以下快捷键在应用任意位置均可使用。

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+P` | 命令面板 |
| `Ctrl+P` | 快速打开文件 |
| `Ctrl+,` | 打开设置 |
| `Ctrl+\` | 切换侧边栏 |
| `Ctrl+Alt+B` | 切换右侧面板 |
| `Ctrl+N` | 新建文档（未命名草稿，保存时另存为转正） |
| `Ctrl+Alt+N` | 在文件树中新建文件 |
| `Ctrl+O` | 打开文件 |
| `Ctrl+Shift+O` | 打开文件夹（工作区） |
| `Ctrl+S` | 保存 |
| `Ctrl+E` | 切换渲染模式（源码 / 实时预览） |
| `Ctrl+G` | 打开知识图谱 |
| `Ctrl+Shift+G` | 打开 Git Graph |
| `Ctrl+Shift+Z` | 插入引用（Zotero） |
| `F11` | 切换专注模式（Focus Mode） |

## 渲染模式切换（Ctrl+E）

- `Ctrl+E` 在「源码」与「实时预览」之间热切换。
- 仅对 Markdown 家族文档生效；其他类型的文档按下后无变化。

## 专注模式（F11）

- 按 `F11` 开关专注模式。
- 开启后，光标所在段落保持清晰，其余段落被淡化，帮助你聚焦当前写作内容；移动光标时高亮区会随之跟随。
- 再次按 `F11` 关闭，恢复全文正常显示。

## 编辑器内快捷键

在编辑器获得焦点时，以下编辑、段落与格式快捷键可直接使用（同样可在命令面板中搜索执行）。

### 编辑

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Z` | 撤销 |
| `Ctrl+Y` | 重做 |
| `Ctrl+X` | 剪切 |
| `Ctrl+C` | 复制 |
| `Ctrl+V` | 粘贴 |
| `Ctrl+A` | 全选 |
| `Ctrl+F` | 查找 |
| `Ctrl+H` | 替换 |

### 段落

以下命令仅对 Markdown 家族文档生效。

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+1` ~ `Ctrl+6` | 设为标题 1 ~ 标题 6 |
| `Ctrl+0` | 设为正文段落 |
| `Ctrl+Shift+8` | 无序列表 |
| `Ctrl+Shift+7` | 有序列表 |
| `Ctrl+Shift+Q` | 引用 |
| `Ctrl+T` | 表格 |
| `Ctrl+Shift+K` | 代码块 |
| `Ctrl+Shift+M` | 数学块 |

（任务列表无默认快捷键，可在命令面板中搜索「任务列表」执行。）

### 格式

以下命令仅对 Markdown 家族文档生效。

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+B` | 加粗 |
| `Ctrl+I` | 斜体 |
| `Ctrl+Shift+`` ` `` | 行内代码 |
| `Alt+Shift+5` | 删除线 |
| `Ctrl+Shift+H` | 高亮 |
| `Ctrl+K` | 插入链接 |
| `Ctrl+Shift+I` | 插入图片 |

（清除格式无默认快捷键，可在命令面板中搜索「清除格式」执行。）

## 注意事项

- 中文输入法（IME）正在组合输入时，回车与快捷键不会误触发命令，可放心打字。
- 段落与格式类命令只对 Markdown 家族文档起作用，对其他语言的文档不会有变化。
- `Ctrl+\` 用于切换侧边栏（让位于 `Ctrl+B` 加粗）；`Ctrl+O` 打开文件、`Ctrl+Shift+O` 打开文件夹，二者勿混。
- 没有列出快捷键的命令（如插入脚注、插入参考文献、切换文档语言、关于、使用教程等）均可在命令面板中搜索执行。
