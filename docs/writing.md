# 写作模式升级：打字机、专注、写作 HUD

为长时间沉浸写作准备的三个辅助：把光标行钉在屏幕中央的**打字机模式**、淡化其余段落的**专注模式**，以及一个可选的**写作 HUD** 悬浮窗（码字速度 / 码字时间 / 番茄钟）。三者都是即时开关、纯内存，不写任何设置文件。

## 打字机模式

开启后，**光标所在的行始终滚动到视口垂直居中**——视线固定在屏幕中央，不必随着越写越往下而低头。文首与文末也能居中（编辑器上下留出半屏空白）。

- 开启：命令面板（`Ctrl+Shift+P`）→「视图：打字机模式」，再执行一次关闭。
- 中文输入安全：合成中文时不滚动，绝不打断输入法（与编辑器其余部分同一套 IME 保护）。

## 专注模式

开启后，**除光标所在段落外的所有行都被淡化**，只有你正在写的这一段保持清晰，随光标移动跟随。

- 开启：`F11`，或命令面板 →「视图：专注模式」。
- 与打字机模式可叠加使用：居中 + 淡化，进入纯粹的写作隧道。

## 写作 HUD

一个左下角的小悬浮窗，给你三个写作指标。**默认关闭**——需要时才开，不打扰。

打开方式（任选其一）：

- Creative（长篇创作）模式下，状态栏右侧有一个「写作 HUD」入口，点一下即开。
- 命令面板 →「写作：写作 HUD（码字速度 / 时间 / 番茄钟）」。

面板内容：

- **码字速度**：最近 60 秒内敲下的字数（即「字/分」），随你停笔自动衰减。粘贴与插入参考文献等程序化写入不计入，中文输入合成期也不计入，反映真实手敲速度。
- **码字时间**：本次打开 HUD 以来累计的写作时长，关掉 HUD 即暂停、重开续计。
- **专注番茄钟**：25 分钟专注 / 5 分钟休息循环。有开始 / 暂停与重置按钮，到点自动切换工作与休息阶段，进度条显示当前阶段剩余。

> 这些指标全部只存在内存里，关闭应用即清零，不会写入磁盘、不进任何统计。
