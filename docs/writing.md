# 写作模式升级：打字机、专注、写作 HUD

为长时间写作准备的三个辅助。**打字机模式**把光标行钉在屏幕中央，**专注模式**淡化其余段落，**写作 HUD** 是一个可选的悬浮窗，显示码字速度、码字时间和番茄钟。三个都是即时开关，状态只放在内存里，不写设置文件。

## 打字机模式

开启后，光标所在的行会滚动到视口垂直居中，视线固定在屏幕中央，不必随着越写越往下而低头。文首与文末也能居中（编辑器上下留出半屏空白）。

- 开启：命令面板（`Ctrl+Shift+P`）→「视图：打字机模式」，再执行一次关闭。
- 中文输入安全：合成中文时不滚动，不会打断输入法（与编辑器其余部分共用同一套 IME 保护）。

## 专注模式

开启后，光标所在段落以外的行会被淡化，只有正在写的这一段保持清晰，并随光标移动。

- 开启：`F11`，或命令面板 →「视图：专注模式」。
- 可以和打字机模式叠加使用，居中加淡化。

## 写作 HUD

左下角的一个小悬浮窗，显示三个写作指标。默认关闭，需要时再开。

打开方式（任选其一）：

- Creative（长篇创作）模式下，状态栏右侧有一个「写作 HUD」入口，点一下即开。
- 命令面板 →「写作：写作 HUD（码字速度 / 时间 / 番茄钟）」。

面板内容：

- **码字速度**：最近 60 秒内敲下的字数（即「字/分」），停笔后自动衰减。粘贴、插入参考文献等程序化写入不计入，中文输入合成期也不计入，统计的是实际手敲的量。
- **码字时间**：本次打开 HUD 以来累计的写作时长，关掉 HUD 即暂停、重开续计。
- **专注番茄钟**：专注 / 休息循环（默认 25 / 5 分钟，可在面板底部两个分钟输入框里自定义，范围 1-180 分钟）。有开始 / 暂停与重置按钮，到点自动切换工作与休息阶段，进度条显示当前阶段剩余。

**拖拽摆放**：按住 HUD 顶部标题栏可把它拖到屏幕任意位置，松手即停；位置在本次会话内记住（关掉再开仍在原处）。

> 这些指标全部只存在内存里，关闭应用即清零，不会写入磁盘、不进任何统计。
