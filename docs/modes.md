# 三模式与工作区

InkStream 用同一个编辑器内核提供 Standard（通用）、Academic（学术）、Creative（长篇创作）三种模式，配合"打开文件夹作工作区"的文件管理，让你在不同写作场景下都拿到合适的面板与工具。

## 三种模式

模式只改变界面布局与右侧面板提供的工具，不改变文件格式，也不会改动文档内容。你随时可以在三种模式间来回切换，正在编辑的文件、标签页与未保存内容都不受影响。

- **Standard · 通用**：日常写作与笔记。右侧面板提供：大纲、反链、局部图谱。
- **Academic · 学术**：论文与带引用的写作。右侧面板提供：引用、Typst 预览、大纲；侧边栏顶部还会多出 Zotero 文献库面板。
- **Creative · 长篇创作**：小说等长篇。右侧面板提供：Codex、场景概要；侧边栏会在文件树上方叠加章节-场景树。

## 切换模式

- 点击窗口右下角状态栏的**模式指示器**（带圆点的模式名），在弹出菜单中选择目标模式，当前模式行尾会有勾选标记。
- 或打开命令面板（Ctrl+Shift+P），执行「模式：切换到 Standard / Academic / Creative」。

模式没有占用全局快捷键，切换一律通过上面两种入口完成。

## 布局与面板

界面分为三栏：左侧边栏（文件树等）、中央编辑区、右侧面板。

- **每种模式各自记忆布局**：你在某个模式下调整的栏宽与折叠状态，会单独保存到该模式；切回来时恢复原样。
- 切换左侧边栏：Ctrl+\（也可点编辑器标签栏最左侧的面板按钮）。
- 切换右侧面板：Ctrl+Alt+B（也可点标签栏最右侧的面板按钮）。
- 重置当前模式布局：命令面板执行「视图：重置当前模式布局」。
- 右侧面板顶部按模式列出对应的标签，点击标签切换；当前没有内容时显示该标签的空态提示。

## 打开文件夹作工作区

InkStream 以"一个文件夹"为工作区（同一窗口同时只有一个工作区）。

1. 在侧边栏空态点击「打开文件夹」，或按 Ctrl+Shift+O，或在命令面板执行「文件：打开文件夹」。
2. 在弹出的系统目录对话框中选择文件夹，确认后该文件夹即成为当前工作区，文件树随之载入。
3. 侧边栏空态还会列出**最近打开**的工作区，点击任一项即可直接重开。

切换到另一个工作区时，若当前工作区是 git 仓库且有未提交更改，会先提示「直接切换 / 提交并切换 / 取消」。切换工作区不会丢失任何文件——已打开的文件仍保留在磁盘上。

## 文件树操作

工作区载入后，侧边栏顶部条显示工作区名称与一排操作图标（新建文件、新建文件夹、折叠全部、刷新）。

- **打开文件**：点击文件名即在编辑器中打开；点击文件夹展开或收起。
- **新建文件**：点顶部「新建文件」图标，或按 Ctrl+Alt+N，然后就地输入文件名。
- **新建文件夹**：点顶部「新建文件夹」图标，就地输入名称。
- **重命名**：选中条目按 F2，或在命令面板对当前文件执行「文件：在文件树中重命名」，就地改名。
- **删除**：选中条目按 Delete，或执行「文件：删除到回收站」（删到系统回收站，可找回）。
- **移动**：在文件树中直接拖拽条目到目标文件夹。
- **折叠全部**：点顶部「折叠全部」图标，或执行「视图：折叠文件树」。
- **刷新**：磁盘有变动时点「刷新」图标重新读取文件树。

注意事项：

- 新建/重命名时，名称不能包含 `/ \ : * ? " < > |` 等字符，否则会提示并退回继续编辑；与已有文件同名时也会退回让你改名。
- 未打开工作区时新建文件/文件夹会提示先打开一个文件夹。
- 以点开头的条目（如 `.git`）默认隐藏。
- 文件树默认全部折叠，展开状态仅在本次会话内有效；重新打开工作区会从全折叠开始。
- 文件名右侧出现小圆点表示该文件有未保存改动。

## 临时打开工作区之外的文件

按 Ctrl+O（或命令面板「文件：打开文件」）可打开任意位置的单个文件：

- 若文件在当前工作区内，按普通文件打开。
- 若文件在工作区之外，会以**非工作区文件**的形式打开，并且**不会切换工作区**。
- 非工作区文件在标签页上有外链图标、文件名显示为斜体，鼠标悬停可看到它的完整路径。

新建文档（Ctrl+N）会开一个未命名草稿，无需先打开工作区即可书写；按 Ctrl+S 时再选择另存为正式文件。

## 外部修改重载

当某个文件在 InkStream 外被其他程序改动、且与你正在编辑的内容不一致时，编辑器顶部会弹出提示条，提供两个选择：

- **重载（丢弃我的修改）**：用磁盘上的最新内容覆盖编辑器，放弃你未保存的改动。
- **保留我的（覆盖磁盘）**：会先二次确认，确认后用编辑器中的内容覆盖磁盘，磁盘上的外部改动将丢失。

在你做出选择之前，该文件的自动保存会暂停，以免误覆盖外部改动。

## 自动保存

- 编辑后会**自动延迟保存**（默认停止输入约 0.5 秒后落盘），无需手动操作。
- 随时可按 **Ctrl+S** 立即保存当前文件。
- 标签页与文件树上的小圆点表示有未保存改动，保存后消失。
- 可在设置中关闭自动保存；关闭后改动仍显示未保存标记，由你用 Ctrl+S 手动保存。
- 未命名草稿没有磁盘路径，不会被自动保存；按 Ctrl+S 另存为文件后才开始自动保存。
