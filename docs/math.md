# 数学公式块

在 InkStream 里直接写公式，离开光标即就地渲染——支持 KaTeX、MathJax 与 Typst 三套引擎，无需切换预览窗口。

## 三种公式块

InkStream 提供三类围栏式数学块，按引擎选择写法，渲染效果各不相同：

- ` ```math `：用 KaTeX 渲染，速度最快，适合常见数学公式。
- ` ```latex `：用 MathJax（SVG 输出）渲染，LaTeX 兼容性更广，公式颜色自动跟随当前主题。
- ` ```typst `：用 Typst 引擎渲染，按 Typst 排版语法书写，结果以白底卡片（类似纸张预览）呈现。

每个块都写在三个反引号围栏内，例如：

````
```math
E = mc^2
```
````

把光标移出块外，公式即自动渲染；把光标移回块内，块会还原成可编辑的源码。

## 用 /math 触发器插入

在编辑器的空白处或空格后键入 `/`，会弹出补全菜单，可快速插入公式块：

- `/math` — 数学公式块（KaTeX）
- `/latex` — LaTeX 公式块（MathJax）
- `/typst` — Typst 排版块

选中后会插入对应的围栏块，并把光标直接放进块内空行，接着输入公式即可。补全菜单里还有 `/code`、`/code-js`、`/code-py`、`/code-rust`、`/code-bash` 等代码块选项。

注意：`/` 触发器只在行首或空白后生效，普通文本（如路径中的 `a/b`）不会误触发。

## 块公式 $$…$$ 与行内公式 $…$

除围栏块外，还可以用美元符写公式，统一由 KaTeX 渲染：

- 行内公式：用单个 `$` 包裹，如 `$a^2 + b^2$`，渲染后随正文行内排版。
- 块公式：用 `$$` 包裹（可跨多行），单独成块、居中显示。

快捷键：按 `Ctrl+Shift+M`（命令面板中为「段落：数学块」）会在光标处插入一对 `$$ $$`，并把光标放在中间，直接开始写。

与三种写法的关系：`$…$`、`$$…$$` 和 ` ```math ` 围栏共用同一套 KaTeX 引擎，渲染结果一致；当光标落在公式所在行时，会显示原始源码方便编辑，移开后恢复渲染。

## 就地编辑与悬浮工具栏

公式渲染后，把鼠标移到公式上会出现一个悬浮工具栏，提供三个按钮：

- 编辑（双栏预览）：打开一个左右分栏面板，左侧编辑源码、右侧实时预览。
- 复制源码：把该公式的源码复制到剪贴板。
- 删除公式块：移除整个公式块。

进入编辑的方式有两种：点击已渲染的公式，或点工具栏的「编辑」按钮。双栏面板里编辑完成后，点「完成」按钮或按 `Esc` 即退出，恢复就地渲染。

## 懒加载说明

为了保证启动速度，三套渲染引擎都按需加载，文档里没有公式时完全不占用资源：

- 只有当文档出现第一个对应类型的公式块时，才会加载相应引擎（KaTeX / MathJax / Typst）。
- 引擎尚未就绪时，公式位置会先显示源码作为占位；加载完成后自动替换为渲染结果，无需手动刷新。
- Typst 块在编辑时有约 200 毫秒的防抖延迟，停止输入后才重新编译渲染。

注意：Typst 引擎相对较重（首次加载需要拉取较大的 wasm 模块），首个 Typst 块出现时可能会有短暂的「编译中」占位，属正常现象。
